# iOS Workshop

This repository contains two parallel iOS projects that cover the same set of workshop topics — one built with **UIKit** and one built with **SwiftUI**. They are kept in sync so you can compare approaches side by side.

| Project | Framework | Xcode Project |
|---|---|---|
| `ios-workshop-uikit` | UIKit | `./ios-workshop-uikit/ios-workshop-uikit.xcodeproj` |
| `ios-workshop-swiftui` | SwiftUI | `./ios-workshop-swiftui/ios-workshop-swiftui.xcodeproj` |


## Opening the UIKit Project

1. Double-click `ios-workshop-uikit/ios-workshop-uikit.xcodeproj` to open it in Xcode.
2. Select a simulator from the scheme toolbar at the top.
3. Press **⌘R** (or click the Run button) to build and launch the app.

The app launches into `RootViewController`, which is a `UINavigationController`-based menu listing all examples.

---

## Opening the SwiftUI Project

1. Double-click `ios-workshop-swiftui/ios-workshop-swiftui.xcodeproj` to open it in Xcode.
2. Select a simulator from the scheme toolbar at the top.
3. Press **⌘R** (or click the Run button) to build and launch the app.

The app launches into `RootView`, which is a `NavigationStack`-based menu listing all examples.
